**Post-ship correction (2026-07-13)** — a same-day review against the actual logged Cycle 2 program surfaced two real misses in the shipped Cycle 3 file, both fixed before he started training it:

Fixed by adding a second core movement to Day 1 (Half-Kneeling Pallof Press, new-to-him, anti-rotation) and Day 2 (Reverse Crunch, new-to-him, flexion) — total now 17 sets/wk, genuinely above C2.
2. **About half the accessories shipped unrotated** (Machine Incline Chest Press, Machine Reverse Fly, Dumbbell Lateral Raise on Day 1; Machine Seated Leg Curl on Day 2; Standing Cable Chest Fly on Day 3; Machine Leg Extension on Day 4) despite this log's Decisions section claiming "normal cycle rotation." Amir's call: the goal of accessory rotation is **perceived novelty** — he should feel the program changed — not physiological necessity, so these should have rotated regardless of the superset-format change. Fixed by rotating five of the six to safe SFR-equivalent alternatives: Incline Chest Press → Cable Chest Press · Reverse Fly → Prone Dumbbell Reverse Fly · DB Lateral Raise → Cable Lateral Raise · Seated Leg Curl → Stability Ball Leg Curl · Standing Cable Chest Fly → Cable Chest Fly. **Machine Leg Extension was kept** — checked the library for a genuine isolated-knee-extension alternative and none exists; forcing a swap would mean a worse exercise just to satisfy the letter of the rule, so it stays until the library has a real alternative.
